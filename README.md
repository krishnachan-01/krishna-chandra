## 🚀 Projects

### 🧠 AI-Based Intrusion Detection System
> Deep Neural Networks + Federated Learning for network threat detection

Built a privacy-preserving IDS that trains across distributed nodes using Federated Learning — so sensitive network data never leaves the source. The DNN model detects anomalies and classifies threats in real time.

### 🔍 Fraud Detection with Hidden Markov Models
> Statistical sequence modeling applied to transaction fraud

Modeled normal user behavior as state transitions using HMMs, flagging deviations as potential fraud. Focused on low false-positive rates to keep the system practical in real-world scenarios.

### 🏭 Enterprise Network Simulation @ BHEL Chennai
> Six-department topology · OSPF · VLANs · Subnetting

Simulated a full enterprise network during my internship at BHEL's DTG division. Designed subnets with /26 masks, configured OSPF with wildcard masks, and implemented SVI-based inter-VLAN routing across departments.
